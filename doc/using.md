# Using

Ranch accepts non-command inputs as expression for compute.

## Operators

> Ranch supports process priority.
>
+ `+` Addition.
+ `-` Subtraction.
+ `*` Multiplication.
+ `/` Division.
+ `^` Exponentiation.
+ `%` Modulo.

## Exmaples

```
Ranch> 5+5+5+5+5
25,00000
Ranch>
```
```
Ranch> 5+5+5+5+5/0
[WARNING FOR ZERO DIVISION]
20,00000
Ranch>
```

```
Ranch> 5.5-5
0,50000
Ranch>
```
